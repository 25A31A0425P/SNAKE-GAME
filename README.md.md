# 🐍 Snake Game in C

This is a simple console-based Snake Game written in C. The game runs in the terminal using basic graphics and keyboard input.

## 🎮 Features
- Classic snake movement (W, A, S, D keys)
- Random fruit generation
- Score tracking
- Game over when hitting walls
- Simple and beginner-friendly code

## 🛠️ Technologies Used
- C Programming Language
- Standard Libraries:
  - stdio.h
  - conio.h
  - windows.h
  - stdlib.h

## 🎯 Controls
| Key | Action |
|-----|--------|
| W   | Move Up |
| S   | Move Down |
| A   | Move Left |
| D   | Move Right |
| X   | Exit Game |
